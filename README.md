# Portfolio - Keerthik T

A professional, responsive portfolio website built with React and Tailwind CSS.

## Features
- **Dark Mode Aesthetic:** Modern dark mode interface with glassmorphism effects.
- **Smooth Animations:** Integrated CSS animations and smooth scrolling.
- **Responsive Design:** Fully responsive layout that looks great on all devices.
- **Modular Architecture:** Organized using Atomic Design principles with separate React components.
- **CDN Powered:** No build step required! Uses Babel standalone to compile JSX directly in the browser for maximum simplicity in this demonstration.

## Technologies Used
- React 18
- Tailwind CSS
- Phosphor Icons
- Babel Standalone

## Getting Started

Simply open `index.html` in your favorite modern web browser. No installation or build steps required.

## Structure
- `index.html` - Main entry point and Tailwind config
- `app.js` - Main React application wrapper
- `components/` - Individual React components for each section
